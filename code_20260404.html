大赛介绍

本次大田一中东部校区第三十四届校园科技文化艺术体育劳动节摄影大赛，正式面向高一、高二全体同学征稿！我们邀请你用镜头捕捉青春瞬间，用光影讲述独家故事。

你可以定格校园里的晨光晚霞、运动赛场的热血拼搏、劳动实践的成长瞬间，也可以记录家乡大田的青山绿水、古厝烟火、特色风物，让每一张照片，都成为青春最鲜活的注脚。

参赛规则

作品要求

  • 参赛作品必须为原创，严禁抄袭、盗用他人作品
  • 作品至少包含1张大田一中东部校区校园风景照
  • 其余作品可围绕大田家乡风光、人文风貌创作
  • 图片清晰，主题积极向上，传递青春正能量

版权须知

  • 严格遵守《中华人民共和国民法典》肖像权相关规定
  • 禁止上传包含他人清晰可辨认肖像的作品
  • 若因违规上传引发肖像权纠纷，由投稿人自行承担全部法律责任
  • 主办方拥有参赛作品的非商业性宣传使用权

作品主题类别

科技主题

捕捉校园科技活动、创新实践、智慧校园场景，用镜头记录科技之美。

文化主题

定格校园文化活动、家乡人文底蕴、传统民俗风貌，传承文化基因。

艺术主题

记录校园艺术瞬间、光影美学、创意构图，用镜头诠释青春艺术气息。

体育主题

捕捉运动赛场的热血拼搏、青春活力，定格挥洒汗水的精彩瞬间。

劳动主题

记录劳动实践的成长瞬间、校园劳动场景，致敬每一份辛勤付出。

家乡风光

定格大田的青山绿水、古厝烟火、特色风物，记录家乡的美好风光。
